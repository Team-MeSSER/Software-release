📖 How to Start

Place Setup_Simmar_102.zip or Setup_Simmar_102eng.zip in any folder and run it.
A C:\Simmar folder will be created automatically.

Please run the setup program with administrator privileges.
Right-click the installer and select "Run as administrator" to ensure proper installation.
If you see a SmartScreen warning, click "More info" and then "Run anyway" to proceed.
